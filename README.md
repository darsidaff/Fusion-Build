# «Фьюжн»

Головоломка для двоих игроков с подключением по локальной сети.

## О проекте

Два игрока попадают в параллельные версии одной и той же комнаты. Им предстоит взаимодействовать с разными версиями реальности, совместно решать необычные головоломки и в итоге воссоединиться, чтобы завершить путешествие.

## Моя роль

- Реализация базового мультиплеерного взаимодействия (подключение игроков, синхронизация состояния)

- Разработка асимметричного игрового процесса для двух игроков

- Реализация механик передвижения и управления персонажем

- Создание и настройка игровых уровней

- Разработка логики головоломок и условий прохождения уровней

- Тестирование сетевого взаимодействия и исправление багов
